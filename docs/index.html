Verification Standard

Five Criteria.
No Exceptions.

Every tool in the RelayForge registry is evaluated against the same public rubric. No black-box scoring. No pay-to-play. This is the standard.

1
Classification
Scope is defined. Limits are documented. You know exactly what this tool can and cannot touch before it runs.
2
Developer Accountability
Named. Reachable. Stands behind it. Anonymous tools don't get verified. Someone's name is on the work.
3
Test Documentation
Show the work, not just the result. Passing tests are the floor. We want to see what you tested and why.
4
Independent Validation
At least one set of eyes that isn't the builder's. Peer review isn't optional — it's how trust transfers.
5
Usability Threshold
A non-technical user can operate it without breaking something. We call this the iOS test. If it requires a manual, it fails.
◈ RelayForge VERIFIED 5/5 Criteria · Trust Score A

This badge means something. It means a tool cleared all five criteria, had independent eyes on it, and was scored by agents before it reached yours. Discovery platform, not a warranty — but as close as this space gets to honest signal.
